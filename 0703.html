# 2주차 수업 내용

  • 싱글 파일 컴포넌트

    • HTML, CSS, JS 코드를 한 파일에서 관리할 수 있음.
    • vue cli 를 이용하여 프로젝트를 생성하면 App.vue 파일을 확인하여 파일 구조 확인 가능
    <template>
    <!-- html 소스 영역 -->
</template>

<script>
    // 자바스크립트 영역
</script>

<style>
    /* CSS 영역 */
</style>
    • 뷰 로더에 의해 html과 css를 자바스크립트 형식으로 파싱하여 html에 표현

  • npm

    • npm이란 전 세계 자바스크립트 라이브러리가 있는 공간
    • npm 설정 파일 생성은
    npm init
    • npm 라이브러리 다운로드
    npm install 라이브러리이름
