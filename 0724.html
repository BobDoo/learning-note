# 5주차 수업 내용

  • 뷰 라우터 옵션
    • routes:라우팅 할 URL과 컴포넌트 값 지정
    • mode:URL의 해쉬 값 제거 속성
    new VueRouter({
  mode: "history",
  routes: [
    { path: "/login", component: LoginComponent },
    { path: "/home", component: HomeComponent },
  ],
});
    • 위 코드는 라우팅을 힐 때에 URL에 #값을 제거하고, 각각 URL값에 해당하는 컴포넌트를 뿌려줌
